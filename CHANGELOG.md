## Features

- Initial release: end-to-end quickstart connecting Foundry Local on Azure Local with the Sovereign Chat Experience starter.
- Default model: Ministral 3 3B (any GPU).
- Alternate model manifests: Ministral (mid-size), Mistral Small, Nemotron (80 GB).
- README-driven 9-step walkthrough, no wrapper code.
